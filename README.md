# 🚕 Crazy Taxi Bot

Интерактивный Telegram-бот для заказа экстремальных и юмористических услуг во время поездки в такси.

## 🚀 Возможности
- 30+ уникальных услуг (от "Сказки на ночь" до "Сжечь машину")
- Интегрированная оплата через Telegram
- Система подтверждения заказов водителем
- Анонимный чат с водителем ("Шепот")
- Юридическая поддержка через AI Адвоката

## 🛠 Установка на Amvera
1. Создайте проект на Amvera.ru.
2. Привяжите этот репозиторий.
3. Добавьте переменные окружения: `API_TOKEN`, `PAYMENT_TOKEN`, `DRIVER_ID`.
4. Нажмите "Собрать и запустить".
